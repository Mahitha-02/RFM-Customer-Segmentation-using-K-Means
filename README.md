RFM Customer Segmentation using K-Means Clustering

Description

Developed a customer segmentation system using RFM (Recency, Frequency, Monetary) analysis and K-Means Clustering. The project analyzes customer purchase behavior and groups customers into different segments to support targeted marketing and customer retention strategies.

Features

- Customer data preprocessing and cleaning
- RFM metric calculation
- Feature scaling using StandardScaler
- Customer segmentation using K-Means Clustering
- Elbow Method for optimal cluster selection
- Cluster visualization using Matplotlib
- Export segmented customer data to CSV format

Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

Dataset Attributes

- CustomerID
- InvoiceNo
- InvoiceDate
- TotalAmount

Workflow

1. Load customer transaction dataset.
2. Clean and preprocess the data.
3. Calculate Recency, Frequency, and Monetary values.
4. Standardize features using StandardScaler.
5. Apply K-Means clustering.
6. Visualize customer segments.
7. Save segmented customer information.

Output

- Customer Segments
- Elbow Method Graph
- Cluster Visualization
- RFM_Segments.csv

Applications

- Customer Retention Analysis
- Targeted Marketing Campaigns
- Customer Value Identification
- Business Decision Support

Conclusion

The project successfully segments customers based on purchasing behavior, helping businesses identify valuable customers and improve marketing effectiveness through data-driven decision-making.
